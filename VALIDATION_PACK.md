# QuoteFlow 首轮验证动作包（48小时）

## 目标
验证：目标用户是否愿意持续使用并愿意为效率付费。

## 核心指标
- 生成率 = generate_success / page_view
- 复制率 = copy_success / generate_success
- 意向付费点击（先用“升级Pro”按钮占位）

## A/B 文案入口
- A版：`https://jijiuaaaaa.github.io/quoteflow/?ab=A`
- B版：`https://jijiuaaaaa.github.io/quoteflow/?ab=B`

## 渠道发布文案（可直接复制）

### 文案1（痛点型）
接单最尴尬的不是报价，是催款。  
我做了个小工具，填4个字段，60秒生成报价+催款文案。  
有需要的试试：{链接}

### 文案2（效率型）
独立开发者的沟通模板自动化：  
输入客户、项目、金额、截止日 -> 直接出可发送文案。  
我在测实用性，欢迎拍砖：{链接}

### 文案3（结果型）
如果一个小工具能让你每单少来回沟通3次，你会愿意付费吗？  
我做了个最小版本，欢迎直接试：{链接}

## 反馈问题（发给首批种子用户）
1. 你会在真实项目里用它吗？（会/不会）
2. 你最常用的是“报价”还是“催款”？
3. 你愿意为这个工具每月付多少钱？
4. 你最希望它补什么功能？

## 执行节奏
- Day1 上午：发布A/B两版入口
- Day1 晚上：导出metrics JSON，比较生成率/复制率
- Day2：按胜出文案继续投放并迭代

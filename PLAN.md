# QuoteFlow（报价单 + 催款助手）立项

## 放弃说明
- 已停止 Excel 911 作为当前主项目（保留代码，不继续投入）。

## 机会来源（社区信号）
- HN近期出现“聊天内发票/账单”类需求，核心痛点是：
  1) 报价/发票流程碎片化
  2) 跟进催款文案难写
  3) 小商家/自由职业者需要轻量工具

## 新项目定位
做一个移动端优先的「报价单生成 + 一键催款文案」工具。

## 目标人群
- 自由职业者（设计/剪辑/开发）
- 小商家与个体服务者
- 通过微信/WhatsApp/Telegram沟通成交的人群

## 今晚MVP
1. 输入客户名、项目名、金额、截止日期
2. 自动生成：
   - 报价单文本（可复制）
   - 温和催款文案
   - 强硬催款文案
3. 一键复制 / 手机端可直接分享

## 商业化
- 免费：每日3次
- Pro ¥19/月：无限生成 + 多语模板 + 历史记录

## 成功标准（首轮）
- 从打开到拿到可发送文案 < 60秒
- 复制按钮点击率 > 35%
- 愿意付费意向（问卷）> 10%
